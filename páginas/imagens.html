Inserindo Uma Imagem Com Link

Comandos Básicos:

Seta aleatória (link p/ comandosbasicos.html)




comando "img" (entre < >)cria uma tag para adicionar uma imagem

atributo src="" localiza as imagens

atributo alt="" adiciona um texto caso a img seja incompativel

atributo widht="" define o tamanho (altura, porem sem o atributo de largura a img cresce proporcionalmente)

comando "a href="">/a" para transformar a img em link("img dentro de href")
